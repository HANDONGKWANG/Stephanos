DDL(Data Definition Language) : 데이터베이스를 정의 ___drop x
create,alter,drop (테이블 생성, 테이블수정, 테이블삭제)

DML(Data Manipulation Language) : 데이터에 대해 조회 수정 삭제
select   insert   update   delete

DCL(Data Control Language) : 권한 부여 취소
grant   revoke

<암기사항>

1. select  컬럼 
2. from 테이블 
3. where 조건
4. group by 컬럼 
5. having 조건 
6. order by 컬럼  
asc/desc(오름/내림차순)

primary key : 레코드의 중복을 방지, 반드시 유일한 값, unll값 허용 안됨,pk
unique : 반드시 유일한 값, 중복방지,null값 허용, uk 
not null : 반드시 값을 가져야 한다. nn
check : 지정한 조건에 일치하는 데이터 값만 저장 가능
